# Mission Control Dashboard (v1)

A multi-company control plane for OpenClaw-driven businesses.

## Quickstart

```bash
pnpm install
pnpm -C apps/api dev
pnpm -C apps/web dev
```

- API: http://localhost:3000 (placeholder)
- Web: http://localhost:5173
